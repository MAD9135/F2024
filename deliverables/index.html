# Deliverables Summary

Prerequisite

You will need your GitHub account (opens new window) for most assignments.

Your GitHub user ID should be the same as your college user ID, and your college email MUST be associated to your GitHub account.

You will be creating Private Github repos for your projects and submitting those links in BS LMS. So, Always remember to invite your instructor to the repo as a collaborator.
